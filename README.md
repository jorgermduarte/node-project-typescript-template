# Node typescript api template

A basic express api project structure made in node with typescript.


## commands
- "build": "tsc",
- "eslint:fix": "eslint src/**/*.ts --fix"
- "eslint:check": "eslint src/**/*.ts"
- "start": "npm run build && node dist/server.js"
    
## Project features

- husky
- eslint
- typescript
- nodejs
- docker
- docker-compose
- dependabot
- github workflow
